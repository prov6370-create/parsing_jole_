💄 Asynchronous Cosmetics Scraper (azi.ua)
Потужний асинхронний інструмент для автоматизованого збору даних з інтернет-магазину косметики azi.ua. Створений для швидкого наповнення баз даних, моніторингу цін або аналізу асортименту.

🚀 Основні переваги
Блискавична швидкість: Завдяки асинхронності (asyncio + aiohttp), парсер обробляє сотні сторінок за лічені секунди.

Інтелектуальна черга: Використання Semaphore запобігає блокуванню IP, імітуючи помірне навантаження на сервер.

Гнучка архітектура БД: Автоматично створює окремі таблиці для кожної категорії товарів у SQLite.

Чистота даних: Вбудована логіка INSERT OR IGNORE гарантує відсутність дублікатів та цілісність інформації.

Повна автоматизація: Скрипт самостійно проходить по дереву категорій та пагінації.

🛠 Технологічний стек
Мова: Python 3.10+

Мережа: aiohttp (асинхронні HTTP-запити)

Парсинг: BeautifulSoup4 (LXML парсер)

База даних: SQLite3

Валідація: re (регулярні вирази для очищення цін та артикулів)
