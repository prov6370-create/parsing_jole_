💄 Asynchronous Cosmetics Scraper | azi.uaВисокопродуктивний асинхронний парсер для повного витягування даних з маркетплейсу azi.ua. Побудований на базі неблокуючого вводу-виводу для досягнення максимальної швидкості без ризику потрапити в бан.⚡️ Ключові фішкиAsynchronous Engine: Використання asyncio та aiohttp дозволяє обробляти тисячі товарів паралельно, що в 10-15 разів швидше за звичайні скрипти.Smart Rate Limiting: Вбудований семафор (Semaphore) контролює кількість одночасних з'єднань, оберігаючи сервер від перевантаження.Dynamic DB Architect: Скрипт на льоту аналізує категорії та розкладає їх у окремі таблиці SQLite.Data Integrity: Жодних дублів — використання UNIQUE ключів та стратегії INSERT OR IGNORE гарантує чистоту вашої бази.🛠 Технологічний стекКомпонентТехнологіяПризначенняRuntimePython 3.10+Основна логікаNetworkingaiohttpАсинхронні HTTP-запитиScrapingBS4 (LXML)Парсинг DOM-дереваStorageSQLite3Надійне збереження даних📊 Об'єкти зборуДля кожного лоту парсер збирає повний пакет даних:✅ Назва (Title)✅ Ціна (Price — автоматичне очищення від валюти)✅ Категорія (Category mapping)✅ Специфікації (Об'єм, країна, тип шкіри тощо)🚀 Швидкий запускПобудовано за принципом "Plug & Play" — просто запусти і збирай профіт.Клонуй репозиторій:Bashgit clone https://github.com/your-username/azi-scraper.git
